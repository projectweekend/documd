## That Stuff


### That route

**POST:**
```
/v1/that
```

**Body:**
```json
{
    "message": "that"
}
```

**Response:**
```json
{
    "message": "that"
}
```

**Status Codes:**
* `200` if successful
* `400` if incorrect data provided
